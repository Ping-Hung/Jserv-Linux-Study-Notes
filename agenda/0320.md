# Notes
## What a C programmer should know about memory
- Bump allocator ≠ Slab allocator
    - Bump allocator:
        - Allocate a large chunck upfront from system , using pointer
          arithmetic to distribute memory in each `allocator_alloc` call.
    - Slab allocator:
        - Manage fix-sized memory blocks in slabs (chunks).
        - Good for large number of fix-sized memory allocations.
